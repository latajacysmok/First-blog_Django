# First-blog_Django
A simple music blog consisting of albums. 
Individual albums consist of the author, year of release, genre, covers and songs.
We have the option of registering for this blog, logging out and logging in.
Add editing and deleting albums and songs if we're the owner.
We can add comments and likes on specific albums.
We also have a user profile that we can edit as the owner.
We can also visit profiles of other users.
